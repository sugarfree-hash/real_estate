# 不動産データ生成・分析アプリケーション (仮称)

## 概要
統計学の手法（重回帰モデルと正規分布）を用いて現実的な不動産ダミーデータを生成し、Web上で分析・可視化を行うアプリケーションです。
バックエンドでのデータ生成から、フロントエンドでのインサイト表示まで構築しています。

## アーキテクチャと技術スタック
* **データ生成ロジック:** Java
* **Webフレームワーク:** Python (Django)
* **データ分析:** Pandas,R
* **データベース:** SQLite
* **フロントエンド:** HTML / CSS

## コア機能と工夫した点
* **統計的根拠に基づくデータ生成 (Java)**
  単なる完全な乱数ではなく、間取り・築年数・駅距離を変数とした重回帰分析のモデル式に、標準正規分布に基づくノイズ（誤差項）を加味することで、相関関係を持つ実践的なモックデータをJSON形式で生成しています。
* **異種言語間のシームレスなデータ連携**
  Javaで生成・格納したSQLiteのデータをJSONで出力し、DjangoのORM経由で取得。PandasのDataFrameに変換することで、スムーズなデータ分析パイプラインを構築しました。SQLiteのデータをより詳細に分析できるようRによる統計解析を実装しました。
* **リアルタイムなデータインサイト抽出 (Python/Pandas)**
  取得したデータセットから「平均家賃」や「家賃と築年数の相関係数」などの統計値を裏側で算出し、ブラウザ上に提示しています。

## 実行手順（ローカル環境）

### 1. データの生成 (Java)
1. `DataGenerator.java` をコンパイルして実行します。
2. プロジェクト直下に `real_estate.db` が生成され、100件のデータが挿入されます。同時に`dummy_estate_data.json`が生成されます。

### 2. Webサーバーの起動と分析画面の表示 (Django)
1. Pythonの仮想環境を有効化し、必要なライブラリをインストールします。
   `pip install django pandas`
2. 以下のコマンドで開発サーバーを起動します。
   `python manage.py runserver`
3. ブラウザで `http://127.0.0.1:8000/` にアクセスしてデータ一覧と分析結果を確認します。
